Terms

tags
Keywords surrounded by angle brackets indicated how the text is displayed
attribute
Provides additional information about an element and are specifed in start tab: (e.g. name="value")
Always quote attribute values. Stick to double quotes unless attribute value contains quotes.

Notes

  1. Preffered encoding is UTF-8
  2. Use 'href' attribute to specify a link
  3. Empty elements such as "br", which indicates a line break, do not need to be closed
